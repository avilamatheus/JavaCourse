## Operadores de atribuição / Assignment operators

```java
a += b; // equals to a = a + b;

a -= b; // equals to a = a - b;

a *= b; // equals to a = a * b;

a /= b; // equals to a = a / b;

a %= b; // equals to a = a % b;
```

## Condicional Ternário / Ternary conditional

```java
(condition) ? (value_if_true) : (value_if_false);

//Example:
int a = (10 > 20) ? 1 : 2;
// a = 2
```
